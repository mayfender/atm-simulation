------------ Description --------------
This is java program simulation of ATM it's just J2SE technology and junit library for test case.



------------ How to build and run --------
First have to build the java source code to jar file any java editor can do.
Then use command below to run (for linux). 
       "java -classpath atmSimulation.jar com.exercise.atm.AtmSimulationMain 10 30"
The last two number in command is initialisation of amount of bank note which first one is amount of $20 and last one is $50.
